# Active-Directory-Lab
Active-Directory-Lab

Overview

This project documents my hands-on practice with Active Directory in a Windows Server lab environment running on VirtualBox.

The goal of this lab was to better understand how enterprise environments manage users, groups, and organizational structures within a domain.

Environment

- Windows Server
- Active Directory Domain Services (AD DS)
- Oracle VirtualBox

Tasks Completed

- Created Organizational Units (OUs)
- Added and managed users
- Configured administrative groups
- Explored Active Directory hierarchy and structure

What I Learned

This lab helped me better understand identity and access management in enterprise environments and how Active Directory is used to organize and control resources.

It also reinforced how cybersecurity connects closely with infrastructure and system administration.


Future Improvements

- Group Policy configuration
- Domain-joined client machines
- User permissions and access control
- Security hardening
- SIEM integration and monitoring
